# rally-node
rally project of node version 

### Development
* install back-end
``` shell
npm install
```
* install front-end(app/client)
``` shell
npm install
npm run build:dll
npm run
```

* import sql table
* change db config(app/server/db/builder.js)

* run server
``` shell
node app.js
```

* flow check(OPTION)
``` shell
npm run flow:start
npm run flow
```
